Awal Plastics




I had the opportunity to work on a large-scale enterprise project for Awal Plastics, a leading signage and display solutions provider in the Middle East. The challenge was to design a complete digital transformation system that streamlined their operations, from sales inquiries and project management to production and delivery tracking.

The system encompassed 3000+ wireframe screens across multiple interconnected modules:


    Finance, Sales, Estimation, Stores, Procurement, Production, HRMS, Projects, Maintenance, Security, IT Admin

Problem :


Awal Plastics’ operations were previously managed using multiple disjointed tools and manual processes. This caused workflow inefficiencies, data silos, and frequent delays in project execution.


My Role & Solution :


    Conducted stakeholder interviews and mapped end-to-end workflows for each department. Created a scalable component-based design system to ensure visual and functional consistency across thousands of screens. Delivered high-fidelity prototypes in Figma with clear interaction flows optimized for desktop and tablet use. Collaborated closely with the development team for smooth design handoff using structured naming conventions and reusable UI components.

This project strengthened my expertise in large-scale B2B SaaS product design, handling complex multi-module systems while prioritizing clarity, usability, and scalability.
